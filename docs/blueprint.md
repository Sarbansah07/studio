# **App Name**: Kasparro AI-SEO Platform

## Core Features:

- AI-SEO Audit: Run an AI-powered SEO audit on a given website.
- Brand Selector: Allow users to select a brand from a list of mocked brands to view its data.
- AI Visibility Score: Display the AI Visibility Score for the selected brand.
- Trust / EEAT Score: Display the Trust / EEAT (Expertise, Authoritativeness, Trustworthiness) Score for the selected brand.
- Non-Branded Keyword Coverage: Display the Non-Branded Keyword Coverage for the selected brand.
- Audit Module Details: Show details for each audit module, including score, key insights, issues/flags, and recommendations.
- AI-Powered Recommendations: Provide AI-generated SEO recommendations based on audit results, leveraging an LLM as a tool that helps decide if or when specific insights need to be incorporated to create comprehensive output.

## Style Guidelines:

- Primary color: Deep purple (#6750A4) to evoke intelligence, sophistication, and technological innovation.
- Background color: Light gray (#F2F0FA), a desaturated version of the primary color, to ensure readability and a modern feel.
- Accent color: Blue violet (#7F52FF), analogous to the primary purple, to highlight key interactive elements and maintain visual interest.
- Headline font: 'Space Grotesk', a sans-serif font that provides a modern and technological feel; body font: 'Inter', for a neutral and readable UI.
- Code font: 'Source Code Pro' for displaying architectural representations.
- Use clear, minimalist icons to represent audit modules and data points.
- Employ a clean, data-dense layout with sensible spacing, typography, and contrast to ensure UX clarity.
